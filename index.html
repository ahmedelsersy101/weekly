تاريخ النفاذ: YYYY-MM-DD

نرحب بك في تطبيق Weekly. تحترم هذه السياسة خصوصيتك وتشرح كيف نجمع ونستخدم ونحمي بياناتك. باستخدامك للتطبيق، فإنك توافق على شروط هذه السياسة.

1. البيانات التي نجمعها

  • بيانات الحساب: البريد الإلكتروني، اسم المستخدم (عند التسجيل).
  • بيانات المهام: مهامك، مواعيدها، أولوية المهام، الوسوم، والملاحظات المرتبطة بها.
  • بيانات الجهاز: نوع الجهاز، إصدار نظام التشغيل (لغرض التحسين والدعم).
  • بيانات الاستخدام: إحصاءات استخدام مجهولة لتطوير المنتج.

2. لماذا نستخدم البيانات

  • تشغيل الميّزات الأساسية (المزامنة، التذكيرات، استعادة الحساب).
  • تحسين جودة التطبيق وتجربة المستخدم.
  • إرسال إشعارات تذكير للمهام (بموافقتك).

3. المزامنة والسحابة

نستخدم خدمات مثل Supabase لتخزين بيانات المستخدم. يتم ربط البيانات بمعرّف المستخدم الآمن (user.id). لا نشارك بياناتك مع طرف ثالث دون موافقتك، باستثناء خدمات البنية التحتية (مزود السحابة) التي تعمل كموفر خدمة تقني.

4. التخزين والاحتفاظ بالبيانات

نحتفظ بالبيانات طالما كان حسابك فعالًا أو حسبما يقتضيه القانون. يمكنك طلب حذف حسابك وبياناتك من خلال التواصل عبر البريد الإلكتروني أدناه.

5. مشاركة البيانات مع الطرف الثالث

قد نستخدم خدمات طرف ثالث (مثل مزودي الإشعارات، التحليلات، و Supabase). هؤلاء مزودون فقط لخدمات فنية ولا يحق لهم استخدام بياناتك لأي أغراض تجارية مستقلة.

6. أمان البيانات

نتخذ تدابير أمنية تقنية وإدارية لحماية بياناتك (تخزين آمن للمفاتيح، استخدام متغيرات بيئية، اتصال مشفر TLS). رغم ذلك، لا يمكن ضمان أمان البيانات بنسبة 100% — ننصحك باتباع ممارسات أمان حسابك.

7. حقوق المستخدم

  • الوصول إلى بياناتك.
  • تصحيح المعلومات الخاطئة.
  • طلب حذف الحساب والبيانات.

8. الاتصال

لأي استفسارات حول الخصوصية أو طلبات حذف البيانات، تواصل معنا عبر: privacy@example.com
